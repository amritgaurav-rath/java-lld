Flow in ATM Machine:
- User comes to atm. -> User, Controller
- User enters Pin using his card. -> Card -> pin will be identified by card no. number<->pin
- User will enter amount to withdraw. Cash withdrawals, Balance enquiry
- Validate if the amount can be withdraw -> if user has X amount in his amount. if yes, proceed, if no, show not possible and reset the machine(Card Eject).
- At proceeding, dispenser will process the amount in x number of a moneyType A, y number of moneytype B and so on till amount is reached.
- we will start by processing with moneyType of higher values first, than lower values until we reach the amount.
